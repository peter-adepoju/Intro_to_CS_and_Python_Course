# Mini-Projects — Week 1

There is no mini-project this week. Week 1 weekend work consists of the
weekend assignment only (see `04_ASSIGNMENTS/week_01/weekend_assignment_01.md`).

Mini-projects begin in **Week 3**, once you have learned functions — they
require enough building blocks to make a substantial standalone project
meaningful. Per the syllabus, mini-projects also appear in Weeks 6, 10, and 12.
