# CS_Python_Semester — Course Package Manifest
## Introduction to Programming Using Python
---

## How to Use This Package

This course package is designed to be worked through from Week 1 to Week 14,
one day at a time, five days per week. Each weekday represents approximately
2 hours of focused learning. Weekends are reserved for longer assignments,
mini-projects, review, and reflection — no new lecture content on weekends.

**Start here:** `08_ENVIRONMENT_SETUP/setup_guide.md`
**Then go to:** `00_SCHEDULE/semester_overview.md`
**Then begin:** `00_SCHEDULE/week_01_schedule.md`

---

## Semester Length

**14 Weeks** | 5 teaching days per week | ~2 hours per teaching day
70 total teaching sessions + 28 weekend work blocks

## 14-Week Curriculum Map

| Week | Title | Core Topics |
|---|---|---|
| 1 | Foundations of Computation | What is CS, Python setup, types, variables, strings, branching |
| 2 | Iteration and Loops | while loops, for loops, range(), break, nested loops, approximations |
| 3 | Functions | Defining functions, parameters, return values, scope, docstrings |
| 4 | Recursion | Recursive thinking, base cases, call stack, Fibonacci, Hanoi |
| 5 | Compound Data — Tuples and Lists | Sequence types, indexing, mutation, aliasing, list methods |
| 6 | Dictionaries and Mutation | Dict operations, keys/values, mutable vs immutable, nested data |
| 7 | Testing, Debugging, and Exceptions | Errors, try/except, testing strategies, assertions |
| 8 | Midterm Review and Assessment | Cumulative review, midterm exam |
| 9 | Object-Oriented Programming I | Classes, `__init__`, methods, `self`, encapsulation |
| 10 | Object-Oriented Programming II | Inheritance, class hierarchies, `super()`, polymorphism |
| 11 | Algorithmic Complexity | Big-O notation, measuring runtime, comparing algorithms |
| 12 | Searching and Sorting Algorithms | Linear/binary search, bubble/merge/quick sort |
| 13 | Data and Program Organization | Modules, file I/O, basic data analysis with Python |
| 14 | Capstone and Final Review | Final project, exam prep, course synthesis |
