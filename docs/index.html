Week 1: Foundations

Types, variables, expressions, strings, input/output, f-strings, and branching.

Recommended learning path

  1. Set up Python. Install Python, Jupyter Notebook, or VS Code.
  2. Read the textbook material. Work through examples slowly.
  3. Open each notebook. Type and modify every example.
  4. Complete the practice scripts. Build fluency with repetition.
  5. Take the quiz. Use mistakes as a review guide.
  6. Finish the weekend assignment. Consolidate the week before moving on.
