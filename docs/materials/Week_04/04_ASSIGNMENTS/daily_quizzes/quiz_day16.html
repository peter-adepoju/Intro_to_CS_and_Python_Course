Q1. In one sentence, what is recursion?

Q2. Every correct recursive function must have which two essential parts?

A) A for loop and a while loop B) A return statement and a print statement C) A base case (directly answerable, no further recursion) and a recursive case (calls itself on a smaller sub-problem) D) A parameter and a default value

Q3. What does this print?

def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n - 1)

print(factorial(5))

A) 25 B) 120 C) None D) RecursionError

Q6. What does this print?

def sum_to_n(n):
    if n == 0:
        return 0
    return n + sum_to_n(n - 1)

Q7. In the step-by-step design process from Chapter 16, which question should you answer FIRST when designing a new recursive function?

Q8. What bug does this function have?

        return 1
    count_digits(n // 10)   # result discarded!
