Q3. What is the difference between lst.sort() and sorted(lst)?

A) .sort() works on any iterable; sorted() only works on lists B) .sort() mutates the list in place and returns None; sorted() returns a new sorted list without changing the original C) They are identical â€” just different syntax for the same operation D) sorted() sorts in descending order; .sort() sorts in ascending order

print(nums)
print(a, b)

A) [1, 2, 3, 4, 5] then 5 1 B) [2, 3, 4] then 5 1 C) [2, 3, 4] then 1 5 D) [1, 2, 3, 4] then 5 1
